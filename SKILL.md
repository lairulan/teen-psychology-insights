### Step 0: 热点分析与选题（必须执行）

**目标**：找到当日最有共鸣的心理学话题，确保内容轻松有趣。

**核心聚焦**：青少年（10-18岁）心理 + 日常生活心理 + 趣味心理学

**搜索热搜榜单：**
```
"青少年心理 [昨日日期] 最新"
"心理学 有趣 [昨日日期]"
"家庭教育 青春期 [昨日日期]"
"亲子沟通 技巧 热点"
"心理学冷知识 热门"
"微博热搜 青少年 心理 昨天"
"知乎热榜 教育 心理"
"小红书 育儿 心理 热门"
```

**热度评估标准：**

| 热度等级 | 指标 | 选题建议 |
|----------|------|----------|
| 5星 | 多平台同时上榜 | 首选，必写 |
| 4星 | 2-3个平台上榜 | 优选 |
| 3星 | 单平台热搜 | 备选 |
| 2星 | 小范围讨论但有代表性 | 结合其他话题 |
| 1星 | 无热度 | 转为趣味心理知识型选题 |

**选题决策逻辑：**
```
IF 热度 >= 4星 AND 与心理/教育直接相关 THEN
    写热点话题的轻松解读
ELSE IF 热度 >= 3星 AND 与日常生活心理相关 THEN
    写生活场景切入的心理小文
ELSE IF 用户指定主题 THEN
    按用户指定主题写作
ELSE
    自动选择"趣味心理知识"型选题（心理实验、冷知识等）
```

**约束：**
- 话题必须能用轻松口吻聊
- 排除过于沉重/极端的案例（重度抑郁自杀等直接建议就医）
- 重点关注：日常场景、有共鸣的小事、有趣的心理现象

### Step 1: 信息搜集

围绕选定话题，用 WebSearch 搜集素材：

```
"[话题关键词] 心理学解释 通俗"
"[话题关键词] 生活中的例子"
"[话题关键词] 实用小技巧"
"[话题关键词] 有趣的心理学实验"
```

整理素材清单：
- **一个生活小故事**：开头用，引发共鸣
- **1-2个心理学小知识**：用大白话解释
- **2-3个实用小建议**：简单、马上能用
- **一句走心的结尾**：温暖收尾

### Step 2: 文章写作

**文章结构（800-1200字）：**

```markdown
# [有趣/温暖/引发好奇的标题]

<!-- IMG_PLACEHOLDER: {主体: "[呼应标题的场景]", 动作/状态: "[状态]", 场景/环境: "[环境]", 风格: "温暖治愈系水彩插画，暖橙浅金色调"} -->

## 开头：生活小场景（150-200字）
- 用"你有没有过这种时候……"或"昨天刷到一个帖子……"开头
- 一个日常的、有共鸣的小场景
- 轻松自然地引出话题

## 中间：心理学小知识（300-400字）
- 用聊天的口吻讲1-2个心理学知识点
- 多用比喻："这就好像……"
- 可以穿插有趣的心理学实验或数据
- 避免任何术语，全部用大白话

<!-- IMG_PLACEHOLDER: {主体: "[场景描述]", 动作/状态: "[状态]", 场景/环境: "[环境]", 风格: "温暖治愈系水彩插画，暖橙浅金色调"} -->

## 实用Tips（200-300字）
- 给出2-3个"马上能用"的小方法
- 每个方法用一句话概括 + 一个具体场景
- 比如："下次孩子说'我不想写作业'，试试先说'嗯，累了吧'而不是'赶紧写'"

<!-- IMG_PLACEHOLDER（可选，内容较长时添加）: {主体: "[Tips场景]", 动作/状态: "[状态]", 场景/环境: "[环境]", 风格: "温暖治愈系水彩插画，暖橙浅金色调"} -->

## 结尾：温暖收尾（100-150字）
- 不说教，不总结
- 用一句温暖的话结束
- 比如："当好父母，不需要完美，需要的是愿意理解。晚安。"
```

**写作要求：**

| 维度 | 要求 |
|------|------|
| 语气 | 像闺蜜聊天，口语化，多用"真的"、"你说是不是"等词增加亲切感；"啊"、"哈"、"哈哈"等语气词克制使用，全篇不超过2处 |
| 长度 | **800-1200字**，绝不超过1500字 |
| 段落 | 短段落，每段不超过3-4行，手机阅读友好 |
| 表达 | 零术语，全部大白话，"依恋理论"要说成"孩子为啥老黏你" |
| 温度 | 温暖但不鸡汤，真诚但不矫情 |
| 互动感 | 多用问句："你有没有发现……"、"是不是很像……" |
| 格式 | 正文不要在开头重复标题；文章结尾不要附参考资料 |

**闺蜜式表达示例：**
- 不说"根据心理学研究表明" → 说"心理学家做过一个特别有意思的实验"
- 不说"建议家长采取积极倾听策略" → 说"下次试试先闭嘴听完，真的有用"
- 不说"青少年处于自我同一性建构的关键期" → 说"这个年纪的孩子，其实自己也不知道自己想要啥"

**配图占位符：**

整篇文章放置 **2-3个** 配图占位符：
- 1个在文章**开头**（标题下方，开场图）
- 1个在中间知识部分之后（必放）
- 1个在Tips部分之后（可选，内容较长时添加）

占位符格式：
`<!-- IMG_PLACEHOLDER: {主体: "[人物或场景]", 动作/状态: "[状态]", 场景/环境: "[环境]", 风格: "温暖治愈系水彩插画，暖橙浅金色调"} -->`

```bash
  --size 1792x1024
```

**正文配图生成：**
